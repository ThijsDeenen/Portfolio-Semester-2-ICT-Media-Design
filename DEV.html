portfolio versie 1

Applicatie met JS + JavaScript Advanced

Click for IQ

Uitleg: In deze applicatie moet je op een brein klikken om hem zogenaamd hard na te laten denken. Hiermee verhoog je je IQ die in dit spel als score wordt gezien. De verschillende upgrades aan de linkerkant van het scherm helpen je sneller je IQ te verhogen. Als je over de upgrades hovert krijg je een uitleg te zien waarin staat wat de upgrade doet. Er zijn verschillende prestaties te behalen die aan de rechterkant van het scherm staan. Als de applicatie wordt opgestart kan de speler zichzelf tekenen en zijn/haar naam eronder zetten.

Reflectie:
Ik heb erg lang gewacht met het maken van deze applicatie. Dit omdat ik liever eerst mijn UEX en SCO opdrachten af wilde hebben. Ik kwam uiteindelijk op het leuke idee om een game te maken waarin je punten verzamelt door de klikken, maar dit dan was geavanceerder maken dan dat het klinkt. Ik heb voor het maken van de applicatie zelf geen verdere hulp gehad van het internet. Ik wilde mijzelf uitdagen en kijken wat ik nou echt helemaal zelf kan maken met javascript. Toen ik hier klaar mee was ging ik kijken wat voor advanced javascript ik aan de applicatie toe zou kunnen voegen. Ik zocht een lijst van advanced javascript onderdelen en keek wat elk onderdeel precies inhield. Ik koos ervoor om met canvas te werken, omdat dit mij het meeste aansprak. Ik zocht tutorials op en kwam uiteindelijk bij de tutorial die ik gebruikt heb om beter te leren te begrijpen wat een canvas kan doen en hoe ik het kan implementeren in mijn applicatie.
